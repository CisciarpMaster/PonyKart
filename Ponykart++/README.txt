# Build and run
- Install MingW 4.8.1 or gcc 4.8.1 as the default compiler
- Install the dependencies (use the same compiler for all the dependencies and the project)
- Compile the project, a Code::Blocks project file is provided for Windows and Linux
- Copy the /media folder in the build/ folder
- Run the game in the build folder, with the required .DLLs / .SOs

# Dependencies : 
- Ogre (1.8.1)
- Bullet 2 (>= 2.81-r2613)
- Boost (1.55.0)
- OgreBullet
