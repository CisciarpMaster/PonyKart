# Build and run
- Install MingW 4.8.1 or gcc 4.8.1 as the default compiler
- Install Ogre 1.8.1's header files
- Compile the project, a Code::Blocks project file is provided.
- Run the game in the build folder (where the dependencies are)

# Dependencies (compiled with MingW 4.8.1, Release) : 
- Ogre 1.8.1
